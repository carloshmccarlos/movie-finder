AI电影搜索网站，用户用自然语言描述电影，AI返回最匹配的电影推荐。

## 视觉设计

### 配色方案 (Dark Theme)

```css
--bg-primary: #0f0f0f;      /* 主背景 - 深黑 */
--bg-secondary: #1a1a1a;    /* 次级背景 - 卡片 */

--text-primary: #ffffff;     /* 主文字 - 白色 */
--text-secondary: #a0a0a0;   /* 次级文字 - 灰色 */

--accent-primary: #ff6b35;   /* 主强调色 - 橙色 */
```

## 组件设计

### SearchBox 搜索框

- 大尺寸文本输入框
- 深色背景 + 微妙边框
- Placeholder 显示示例描述
- 支持多行输入
- Enter 键触发搜索

### FilterChip 筛选标签

- 默认状态: 透明背景 + 边框
- 选中状态: 强调色背景
- Hover: 轻微高亮
- 支持单选/多选模式

### MovieCard 电影卡片

- 海报图片 (2:3 比例)
- 标题 + 年份
- 评分 (星星图标)
- 匹配度标签 (高/中/低)
- 观看平台图标
- Hover 效果: 轻微放大 + 阴影

### LoadingState 加载状态

- 居中显示
- 动画 spinner
- 文案: "正在搜索电影..."

### EmptyState 空状态

- 居中显示
- 图标 + 文案
- 示例搜索建议

## 响应式设计

### Desktop (>= 1024px)
- 搜索框宽度: 600px
- 电影卡片: 4列网格
- 筛选栏: 单行显示

### Tablet (768px - 1023px)
- 搜索框宽度: 100%
- 电影卡片: 3列网格
- 筛选栏: 可折叠

### Mobile (< 768px)
- 搜索框宽度: 100%
- 电影卡片: 2列网格
- 筛选栏: 抽屉式展开

## 交互设计

### 搜索流程

1. 用户输入描述 → 实时字数统计
2. 选择筛选条件 → 即时视觉反馈
3. 点击搜索/按Enter → 显示加载状态
4. 结果返回 → 平滑过渡显示卡片
5. 无结果 → 显示空状态 + 建议

### 动画

- 页面切换: fade + slide (200ms)
- 卡片出现: stagger fade-in (50ms间隔)
- 按钮hover: scale(1.02) + shadow
- 筛选切换: 背景色过渡 (150ms)

## 文案规范

### 标题
- 主标题: "AI电影搜索"
- 副标题: "描述你想看的电影，AI帮你找到它"

### 按钮
- 搜索: "找电影" / "搜索"
- 重置: "清除筛选"

### 状态文案
- 加载中: "正在搜索电影..."
- 空结果: "没有找到匹配的电影，试试其他描述？"
- 错误: "搜索出错了，请稍后重试"

### 匹配度
- 高: "高度匹配"
- 中: "中度匹配"
- 低: "可能相关"

### 新增组件

#### MoviePoster 海报组件
- 接收: movieTitle, year, fallbackEmoji
- 功能: 从TMDB获取海报URL
- 状态: loading → loaded / error
- 错误回退: 显示渐变背景 + emoji

#### MovieDetail 详情页组件
- 路由: /movie/:id
- 数据: 从URL state或重新搜索获取
- 功能: 显示完整电影信息
- 操作: 返回、在线搜索、复制信息

#### ExternalSearchButton 外部搜索按钮
- 功能: 检测IP地区，跳转搜索引擎
- 中国IP: Bing
- 其他: Google
- 搜索词: 电影名 + "在线观看"
